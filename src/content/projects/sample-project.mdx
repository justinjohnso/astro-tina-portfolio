# Sample Project

This is a sample project that demonstrates the new Astro + TinaCMS architecture. The content is now stored as Markdown files and managed through TinaCMS.

## Features

- Git-based content management
- Simplified architecture
- Better performance
- Easier deployment

## Technical Details

This project showcases how we've migrated from PayloadCMS to TinaCMS for a cleaner, more maintainable portfolio website.

The new setup provides:
- Faster build times
- Simpler deployment process
- Better developer experience
- Version-controlled content
