# Arduino Weather Station

A comprehensive IoT weather monitoring system that collects environmental data and provides real-time visualization through a web dashboard.

## Hardware Components

- **Arduino ESP32** for processing and WiFi connectivity
- **DHT22** temperature and humidity sensor
- **BMP280** barometric pressure sensor
- **Light sensor** for ambient light measurement
- **OLED display** for local data display

## Software Architecture

The system consists of three main components:

### Sensor Data Collection
The Arduino continuously reads sensor data and processes it locally before transmission.

### Data Transmission
Environmental data is sent via WiFi to a cloud database every 5 minutes, with local caching for offline periods.

### Web Dashboard
A responsive web interface displays real-time and historical data with interactive charts and analytics.

## Key Features

- **Real-time monitoring** of temperature, humidity, pressure, and light levels
- **Historical data tracking** with 1-year data retention
- **Alerts system** for extreme weather conditions
- **Mobile-responsive dashboard** for monitoring on any device
- **Low power consumption** with sleep modes between readings

## Data Visualization

The dashboard includes:
- Live gauge displays for current conditions
- Time-series charts for trend analysis
- Weather pattern recognition
- Export functionality for research purposes

## Impact

This project demonstrates the practical applications of IoT technology in environmental monitoring, providing valuable insights into local microclimates and weather patterns.
