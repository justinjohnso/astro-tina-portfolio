The Tangible Music Looper (DJ Remix Box) is an intuitive physical interface for real-time music remixing that triggers looping audio files when translucent 'blocks' are placed on a board.

Developed for ITP Intro to Physical Computing alongside Michelle Xu, the project evolved from an Arduino MKR Zero prototype to using a Raspberry Pi for multi-threaded audio playback. The system uses hall effect sensors beneath the board to detect magnet-embedded 3D-printed blocks. Each region of the board corresponds to a different stem or audio loop, allowing users to intuitively build up and break down tracks by simply placing or removing blocks.

The instrument features built-in NeoPixel LED feedback to provide visual cues and is completely standalone, utilizing an internal audio amplifier and speakers so no laptop is required for performance.
